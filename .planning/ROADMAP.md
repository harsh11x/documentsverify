# ROADMAP

## Phase 1 - Foundation and Architecture
Goal: establish monorepo, infra, DB schema, taxonomy, and environment baseline.
Requirements: ORG-01, ORG-03

## Phase 2 - Smart Contracts
Goal: implement org and certificate registries with tests and deployment readiness.
Requirements: CERT-02, CERT-03

## Phase 3 - Backend APIs
Goal: auth, org approval, branch, issue, bulk queueing, verify, revoke, search, audit APIs.
Requirements: ORG-01, ORG-02, ORG-03, CERT-01, CERT-02, CERT-03, VER-01, VER-02, VER-03

## Phase 4 - Certificate Rendering
Goal: HTML template, QR generation, and PDF persistence.
Requirements: PDF-01, VER-02, VER-03

## Phase 5 - Org Dashboard
Goal: organization onboarding and certificate management interfaces.
Requirements: ORG-01, ORG-03, CERT-01, CERT-03

## Phase 6 - Public Verification Frontend
Goal: user-friendly verification UI and mobile QR route.
Requirements: VER-01, VER-02, VER-03

## Phase 7 - Super Admin Panel
Goal: organization moderation, taxonomy management, and audit access.
Requirements: ORG-02

## Phase 8 - Security and Launch
Goal: security testing, load/perf validation, compliance checks, and production rollout.
Requirements: All v1
